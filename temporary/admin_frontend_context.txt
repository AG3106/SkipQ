WHAT THE ADMIN CAN DO RIGHT NOW (3 API endpoints)
================================================================================

1. VIEW PENDING CANTEEN REGISTRATIONS
   GET /api/admin/canteen-requests/
   → Returns list of canteens with status "UNDER_REVIEW"
   → Each canteen has: id, name, location, manager info, opening/closing times

2. APPROVE A CANTEEN
   POST /api/admin/canteen-requests/<id>/approve/
   → Moves canteen from UNDER_REVIEW → ACTIVE
   → Returns success message + updated canteen object

3. REJECT A CANTEEN
   POST /api/admin/canteen-requests/<id>/reject/
   → Body: { "reason": "..." }  (optional)
   → Moves canteen from UNDER_REVIEW → REJECTED
   → Returns success message + updated canteen object
